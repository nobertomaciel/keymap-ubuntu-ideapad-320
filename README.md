Keyboard configuration files (KeyMap) for Lenovo IdeaPad 320 notebooks (14IBK, 14ISK, 15IKB, and 15ISK).
Fixing Delete and Insert keys issue on Linux Ubuntu.

Place the files in the correct directories:

* Files in the "symbols" directory: /usr/share/X11/xkb/symbols
* Files in the "keycodes" directory: /usr/share/X11/xkb/keycodes

See the Ubuntu Community Help Wiki for more details:
* https://help.ubuntu.com/community/Custom%20keyboard%20layout%20definitions?action=show&redirect=Howto%3A+Custom+keyboard+layout+definitions

For windows, download portcode.exe from Lenovo website:
* https://support.lenovo.com/iq/pt/downloads/ds543049-portcode-keyboard-patch-for-brazil-for-windows-10-64-bit-ideapads-thinkpads-and-legion-notebooks
